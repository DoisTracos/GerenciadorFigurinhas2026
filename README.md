É um trabalho de faculdade, então se trata de uma grande brincadeira.
Os arquivos ".class" são aparatos para simplificar o código. Nota-se que sem eles, o gerenciador não irá funcionar corretamente.
O arquivo "figurinhas_completo.csv" é a lista completa de figurinhas do álbum, na qual o algoritmo irá se basear.
Automaticamente, o gerenciador irá criar uma lista de figurinhas obtidas, atualizada conforme estas forem cadastradas.
